- **Fichier** : `AirQualityUCI.csv`
- **Description** : Données de qualité de l'air collectées par des capteurs chimiques

## 🚀 Comment exécuter le projet

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/YOUR_USERNAME/Projet_PHEI_Groupe7.git
   ```
2. Ouvrir Anaconda et lancer Jupyter Notebook
3. Naviguer vers le dossier du projet
4. Ouvrir `projet5_calibration_UCI.ipynb`
5. Exécuter les cellules dans l'ordre (**Kernel → Restart & Run All**)
